Proces

Ik wilde erg graag AngularJS gebruiken voor het maken van deze app, maar ik kon nergens een geschikte REST API vinden. Ik besloot daarna om zelf een API te maken en deze zelf te hosten, maar ik liep tegen allerlei proxy/firewall problemen aan.

Nadat ik enkele uren besteed heb om uit te vinden hoe ik dit moest fixen, besloot ik een andere weg te kiezen om dit project zo goed mogelijk af te ronden en zo toch een inkijkje te kunnen geven in mijn vaardigheden.

De data is statisch, maar kan door middel van x-Editable (in combinatie met jQuiry UI) gewijzigd worden en vervolgens opgeslagen worden. Ik heb gebruik gemaakt van het Flexbox Grid systeem en heb vervolgens Isotope gebruikt voor het verwijderen en toevoegen van nieuwe medewerkers. Grunt zorgt ervoor dat alles geminificeerd wordt en ik heb alles met de terminal gecommit naar de GitHub repo.

Verbeterpunten: de datumpicker zou makkelijker in gebruik moeten zijn (format en jaar kiezen), eerder in het proces moeten kiezen tussen Flexbox Grid of Isotope (werken niet helemaal goed samen),
