Medewerkers

Hieronder kun je de medewerkers van Informaat aanpassen en verwijderen door middel van het klikken op de naam, functietitel, geboortedatum en biografie. Dit heb ik gemaakt met x-editable in combinatie met jQuery UI.

Process

I wanted to make this app with Angular JS, but was unable to find a proper REST API to give me the correct data. After that, I wanted to make my own API, but ran into some problems with my firewall/proxy settings.

After spending some time figuring out how to fix this and make it work, I decided to take another route in finishing this project. The data is static HTML but can be changed with the x-Editable functionality.
